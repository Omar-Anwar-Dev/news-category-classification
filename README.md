# News Category Classification

End-to-end multi-class text classification system that assigns one of 42 news categories to an article using its headline and short description.

Built as the deliverable for an academic Data Science course (Level 4) by a 7-person student team over ~4 weeks. Ships with six classical models, a RoBERTa-embeddings + Linear-SVM bonus, a Groq-hosted LLM that explains predictions, and a Gradio demo UI.
